## 📌 Project Overview

AI salaries vary significantly depending on factors such as experience, education, technical skills, company size, industry, and hiring urgency.

This project builds an end-to-end **Machine Learning Regression Model** to predict AI job salaries based on candidate and company attributes.

The project demonstrates the complete Machine Learning workflow including data preprocessing, feature engineering, model training, evaluation, and salary prediction.

---

# 🎯 Problem Statement

Develop a Machine Learning model capable of predicting AI job salaries using multiple job-related and candidate-related features.

The objective is to understand the factors influencing salaries and build an accurate predictive model.

---

# 📂 Dataset

The dataset contains AI job listings with information including:

* Job Title

* Experience Level

* Education Level
* Python Skill
* SQL Skill
* Machine Learning Skill
* Deep Learning Skill
* Cloud Skill
* Hiring Urgency
* Job Opening Count
* Salary (Target Variable)
